# Grade 9 HTML Forms – Lesson Pack

This pack mirrors the 3‑hour workshop:
- 01_intro.html – GET vs POST, name=value
- 02_basic_form.html – minimal forms
- 03_input_types.html – common inputs
- 04_select_textarea_datalist.html – lists and multi‑line text
- 05_validation_css.html – native validation + CSS
- 06_js_enhancements.html – show/hide password, live counter
- 07_formdata_playground.html – submit to page using FormData (no server)
- assets/style.css – shared styling
- assets/playground.js – shared JS helpers

Open any file in a browser by double‑clicking it.
